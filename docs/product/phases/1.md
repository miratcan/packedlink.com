# PackedLink Focus — Phase 0: Stealth SEO + Landing

## Goal
- **Accumulate SEO traffic** → 500+ blog posts indexed on Google before app launch
- **Silent development** → Publish blog without announcing, develop product quietly
- **Ready traffic at launch** → SEO traffic from day one

## Scope

### Landing Page
- Homepage: Explain what it is + "Coming Soon" message
- Header navigation: `Home | Blog | Coming Soon`
- Email capture (optional)
- Warm terracotta/brown design (current design)

### Blog System (Notion → Next.js)
- Write blog in Notion
- `npm run sync-blog` → Downloads to Markdown in Next.js
- Static site generation (build time)
- Blog routes: `/blog` (list), `/blog/[slug]` (detail)
- SEO metadata (title, description, OG tags)
- Automated sitemap.xml
- Google Search Console setup

### Blog Content Strategy
- **Target:** 500 blog posts (over 3 months)
- **Tempo:** Publish 5-10 posts per day (slow indexing)
- **Topics:**
  - Link collection use cases
  - Content curation tips
  - Content creator guides
  - SEO & social media advice
- **Format:** AI-assisted + human editing

### Silent Development (Parallel)
- OAuth login (Google)
- Builder (add, edit, delete links)
- Public list view (/u/[username]/[slug])
- Profile (photo, bio, cover - [ZEN-PROUD])
- Analytics (basic event tracking)

## Success Criteria

1. **500+ blog posts indexed** (verify in Google Search Console)
2. **Blog traffic started** (at least 100 organic visitors/month)
3. **Core MVP ready** (OAuth + Builder + Public list)
4. **[ZEN-PROUD] compliant** (User profile customization)

## Guardrails

- **[ZEN-SEO-COMPOUND]:** Blog system comes before app development
- **[ZEN-CORE-FIRST]:** Create list + share flow must be flawless
- **[ZEN-SOLO-FIT]:** Simple enough for one person to execute
- **[ZEN-MEASURE-FIRST]:** Event tracking ready, data-driven decisions

## Anti-Patterns

❌ Don't tell anyone (no social media sharing)
❌ Don't force email list (keep it optional)
❌ Feature creep (stay focused on core)
❌ Neglect blog (SEO momentum is critical)
